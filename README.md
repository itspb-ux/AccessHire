## üß© Problem Statement

Persons with Disabilities (PwD) face significant barriers in accessing employment opportunities due to:

- Inaccessible job portals
- Non-inclusive application processes
- Lack of assistive features such as screen reader support
- Limited access to skill development resources
- Employers lacking simple tools to adopt inclusive hiring practices

At the same time, employers‚Äîespecially MSMEs‚Äîstruggle to find verified, skilled PwD candidates and manage inclusive hiring efficiently.

---

## üí° Our Solution

**AccessHire** is an accessibility-first hiring platform that bridges this gap by providing:

- Inclusive, accessible job discovery and application workflows
- Dedicated dashboards for both candidates and employers
- Built-in accessibility features such as **High Contrast Mode**,**Voice Search**  and **Read Aloud**
- Skill training modules to improve employability
- Firebase-powered authentication and database for secure and scalable data handling

Accessibility is embedded **by default**, not added later.

---

## üìå Project Description

AccessHire enables Persons with Disabilities (PwD) to:
- Discover jobs easily
- Apply without accessibility barriers
- Track applications
- Improve skills through structured training

Employers can:
- Post jobs
- View and manage applicants
- Update application statuses through a simple dashboard

The platform ensures equal access, usability, and dignity for all users.
